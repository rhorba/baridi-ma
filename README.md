# Baridi.ma — Cold-Chain Logistics Tracker

Morocco exports 3B MAD of perishables annually. No platform tracks temperature-sensitive shipments end-to-end.

## Problem
Pharma distributors, dairy cooperatives, and agri-exporters lose product quality and face EU compliance failures because cold-chain integrity is logged on paper.

## Solution
Real-time cold-chain tracking SaaS — IoT temperature/humidity sensor integration, live alerts, tamper-proof compliance PDF export for EU customs.

## Stack
Next.js 15, TypeScript, PostgreSQL 16 + TimescaleDB, MQTT broker, Docker, Tailwind v4, CMI payments

## Completes
Wassalha (COD parcels) + track.ma (fleet GPS) + Mawsim (agri sellers need cold-chain proof)

## Key Roles
Shipper | Carrier | Receiver | Admin
